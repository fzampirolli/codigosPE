O que é um Desvio Condional?¶

  • O desvio condicional a mais simples entre as estruturas lgicas no sequenciais em lgica de programao e fundamental para o entendimento de fluxo de cdigo.

  • A analogia bsica com o processo de tomada de decises ocorre quando imaginamos um cenrio que proporciona duas possveis alternativas de curso:

    • Se [condio] ento faa [caminho caso verdadeiro] seno [caminho caso falso]

  • Exemplos:

    • Se [est chovendo] ento [resolver palavras cruzadas] seno [andar de bicicleta]
    • Se [ quarta] ento [comer feijoada]

Condições com Lógica Booleana¶

  • O resultado de um teste condicional sempre resultar em um valor booleano, isto :

    • com dois resultados possveis: verdadeiro ou falso
    • Por convenso: $True=1$ e $False=0$

  • Portanto, para condies, sempre usaremos combinaes de operadores lgicos e relacionais para verificar o estado das variveis verificadas. O seguinte pseudocdigo exemplifica algumas condies:

se vai chover, ento leve um guarda-chuva.

se  feriado, ento fique em casa.

se estou atrasado e est chovendo, ento chame um taxi.

se minha nota  menor que 5, ento fiquei de recuperao.

  • Note que para todas as condies acima, a resposta para a condio sempre: verdadeiro ou falso.

  • Caso a condio seja verdadeira, ser executada a operao ou operaes especificadas na sequncia.

  • Codificando-se, as condies tomam a forma se (condio) { comandos }, como exemplos:

se (vai_chover) { leve um guarda-chuva }

se (feriado) { fique em casa }

se (atrasado e chovendo) { chame um taxi }

se (nota<5) { escrever ficou de Recuperao }

Esse cdigo anterior fica melhor se usar a organizao apresentada no captulo anterior:

se (vai_chover) 
  leve um guarda-chuva 

se (feriado) 
  fique em casa 

se (atrasado e chovendo)  
  chame um taxi 

se (nota<5) 
  escrever ficou de Recuperao

  • Observe que foram substitudas as chaves { e }, que definem os escopos das condicionais, pela tabulao. Isso possvel quando o bloco tem apenas uma instruo.

  • Relembrando o Cap. 1 - Fundamentos, podemos usar nas condicionais, variveis booleanas e operadores:

    • relacionais: == ( igual a) != ( diferente de) > ( maior que) < ( menor que) >= ( maior ou igual a) <= ( menor ou igual a)
    • Alm de lgicos: && (e), & (e bit-a-bit) || (ou), | (ou bit a bit) ! (no lgico ou complemento) ~ (complemento bit-a-bit) ^ (ou exclusivo XOR bit-a-bit) << N (shift-left, adiciona N zeros direita do nmero em binrio) >> N (shift-right, elimina N dgitos a direita do nmero em binrio)

  • Os operadores lgicos, como os relacionais, sempre resultaro verdadeiro (V) ou falso (F), porm os operandos tambm so booleanos.

  • Para as condies contendo AND, OR e XOR (ou exclusivo), as tabelas verdade para os dois operandos esquerda e direita, com valores lgicos representados na primeira linha e primeira coluna, so:

| &&| V | F | |:-:|:-:|:-:| | V | V| F | | F | F | F |

|||| V | F | |:-: |:-:|:-:| | V | V | V | | F | V | F |

| ^ | V | F | |:-:|:-:|:-:| | V | F | V | | F | V | F |

  • Logo,
    • os operandos devem ser ambos verdadeiros para que a operao AND retorne verdadeiro,
    • ao menos um deles verdadeiro para que o OR retorne verdadeiro e
    • ambos diferentes para que o XOR (ou exclusivo ^) retorne verdadeiro.

  • Estas expresses, quando combinadas, resultaro sempre em um valor booleano V ou F, que pode ser ento introduzido em um desvio condicional visando realizao de um subprograma.

  • Por exemplo, usando os valores X=1, Y=2 e Z=4, qual o resultado das expresses abaixo?

  1. (X>0 && Y<2)
  2. (Z>0 || Z<5 && Y==4)
  3. (X>>1 ==0 || Y<<2>100)
  4. (X!=0 && Y!=0 && Z<0)
  5. (X=1)
  • Dada a precedncia de operadores estudada anteriormente e a combinao apresentada acima, apenas as expresses 2 e 3 resultariam em verdadeiro, dado $Z>0$, assim como $1>>1$ (shift direita uma casa) em binrio 0; so ambas condies suficientes para que o resultado seja verdadeiro.

Exemplo 01 - Uso de Condicionais Simples¶

  • Digamos que, como exemplo, desejamos calcular as razes da equao de segundo grau usando a funo delta() introduzida no captulo anterior.

  • Sabemos que as razes dependem do sinal do $\Delta$.

  • Logo, a soluo de uma equao do segundo grau se d resolvendo as seguintes condies:

  1. Se $\Delta<0$, $x$ no possui razes reais;
  2. Se $\Delta=0$, $x$ possui duas razes reais idnticas;
  3. Se $\Delta>0$, $x$ possui duas razes reais e distintas;
  4. Calcule as razes, se existirem, usando as equaes:
$$x1 = -b+\frac{\sqrt{\Delta}}{2a}$$$$x2 = -b-\frac{\sqrt{\Delta}}{2a}$$
  • Veja uma soluo em pseudocdigo:
# a funo  definida a seguir
funo delta(a, b, c )
  retorne b * b - 4 * a * c

# programa principal

escreva("Calcula as razes de equao de 2 grau: ax2 + bx + c")
real a = leia("Entre com o primeiro termo a: ")
real b = leia("Entre com o segundo  termo b: ") 
real c = leia("Entre com o terceiro termo c: ") 

# PROCESSAMENTO E SADA
real d = delta(5, -2, 4)

%%writefile cap03exem01.c
#include <stdio.h>
#include <math.h>
float delta(float a, float b, float c) {
  float d = b*b-4*a*c;
  return d;
}

float leia() {
  float valor;
  printf("Entre com um valor: ");
  scanf("%f", &valor);
  return valor;
}

int main(void) {

  // ENTRADAS
  float a, b, c;
  a = leia();
  b = leia();
  c = leia();
  
  // PROCESSAMENTO e SADA
  double d = (double) delta(a, b, c);
  printf("Delta = %.1f\n", d);
  if (d < 0) {
      printf("A equao no possui razes reais");
  }
  if (d == 0) {
      printf("Raz: %.1f",(-b + sqrt(d) / 2 * a));
  }
  if (d > 0) {
      printf("Raz: %.1f e %.1f ",(-b - sqrt(d) / 2 * a), (-b + sqrt(d) / 2 * a));
  }
  return 0;
}

Comando switch¶

O switch verifica se uma varivel (do tipo int ou char) ou no igual a certo valor constante (valor1, valor2, ..., valorN na sintaxe a seguir):

switch (varivel) {
  case valor1:
    Comandos;
    break;
  case valor2:
    Comandos;
    break;
  case valorN:
    Comandos;
    break;
  default: // opcional, caso no ocorram os casos anteriores
    Comandos;
}

Esse comanto switch til quando se tem um menu de opes a ser executado.

%%writefile cap03exem03.c
#include <stdio.h>
int main() {
	float area;
	int escolha, num;
	printf("1. Circulo\n");
	printf("2. Quadrado\n");
	printf("Escolha:\n");

	scanf("%d", &escolha);
	
	switch (escolha) {
	case 1: {
		printf("Raio:\n");
		scanf("%d", &num);
		area = 3.14 * num * num;
		printf("Area do circulo: ");
		printf("%.2f\n", area);
		break;
	}
	case 2: {
		printf("Lado:\n");
		scanf("%d", &num);
		area = num * num;
		printf("Area do quadrado: ");
		printf("%.2f\n", area);
		break;
	}
	default:
		printf("Escolha Incorreta!\n");
	}
	return 0;
}
