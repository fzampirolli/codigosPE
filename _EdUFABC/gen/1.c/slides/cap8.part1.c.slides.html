Introdução¶

  • Ponteiros so variveis especiais que recebem valores referentes endereos da memria principal do computador (RAM - Random Access Memory).

  • Ao ligar um computador, o sistema operacional carregado da memria secundria para a RAM.

  • Quando criamos um programa em alguma linguagem de programao e executamos, esse programa tambm carregado na RAM.

  • Uma varivel x=10 criada nesse programa tambm ser enderaada na RAM, por exemplo, no endereo FF10AF (em hexadecimal).

  • Algumas linguagens de programao aceitam tambm essas variveis especiais do tipo ponteiros. Por exemplo, em C podemos criar int *p=FF10AF.

  • Em geral, no precisamos saber qual o endereo de memria de uma varivel, mas importante saber associar a um ponteiro. Por exemplo, a seguir criada uma varivel inteira x=10 e em seguida um ponteiro p associado a x, incluindo um prefixo &, para pegar o endereo de memria de x e associar a p, com p = &x. Para alterar o contedo de p (o mesmo de x), bastar incluir o prefixo *, ou seja, *p = 15. Ver tambm a figura a seguir para melhor visualizar operaes com ponteiros.

  int x = 10;
  int *p;     // cria um ponteiro para um inteiro
  p = &x;     // faz p apontar para o endereo de x
  *p = 15;    // altera o contedo de p (e tambm de x)
  printf("x=%d *p=%d", x, *p);

Alocação estática¶

  • A alocao esttica ocorre em tempo de compilao a depender a linguagem de programao escolhida. Por exemplo, no exemplo a seguir, o vetor v criado contendo 3 elementos inteiros. No possvel incluir um quarto elemento nesse vetor quando executamos o progroma (ou seja, em tempo de execuo).
  • Ao criar vetores ou matrizes, estamos criando ponteiros para o primeiro elemento dessas estruturas. Por exemplo,
  int v[3] = { 5,6,7 };
  int* p1;
  p1 = v; // observe que aqui no precisa usar &v
  for (int i = 0; i < 3; i++)
    printf("%d %d %d\n", *(p1 + i), p1[i], v[i]); // *(p1+i) = p1[i] = v[i]

  • Assim, nesse exemplo possvel fazer operaes de ponteiros (p1 + i) para acessar os endereos p1, p1+1, p1+2, $\cdots $

  • Para a varivel ponteiro do tipo char, char *p ocupa um byte. Para int so 4 bytes e para double so 8 bytes, a depender da arquitetura do computador.

  • Para saber a quantidade de bytes de uma varivel/tipo, usar a funo sizeof.

  • Por exemplo, para uma varivel do tipo double, basta fazer sizeof (double), ou sizeof(p1), onde p1 foi definido no exemplo anterior.

  • Tambm possvel calcular o tamanho de um vetor alocado estaticamente. Para o exemplo anterior, basta fazer: sizeof(v) / sizeof(int).

  • Esse clculo til para saber o tamanho de um array porm tem algumas restrioes, ver prxima seo.

  • Tambm possvel fazer operaes de ponteiros para array de struct. Ver um exemplo a seguir:

Exemplo para criar um array de struct Aluno contendo 4 atributos lidos do teclado com scanf. Associar um ponteiro para essa struct.

%%writefile cap8ex01.c
#include <stdio.h>
#include <string.h>

typedef struct {
  char nome[50];
  int idade;
  char rua[50];
  int numero;
} Aluno;

int main() {
  Aluno Alunos[2]; // instancia uma vetor do tipo Aluno
  Aluno *p; // cria um ponteiro para Aluno
  p = Alunos; // ATENO: associa p a array de alunos
  // ENTRADA DE DADOS
  for (int i = 0; i < 2; i++) {
    printf("Entre com os dados: nome, idade, rua, nmero:\n");
    fflush(stdin);
    fgets(p[i].nome, 50, stdin);
    scanf("%d", &p[i].idade);
    fflush(stdin);
    fgets(p[i].rua, 50, stdin);
    scanf("%d", &p[i].numero);
  }
  // SADA DE DADOS
  for (int i = 0; i < 2; i++) {
    printf("nome: %s\nidade: %d\n", p[i].nome, p[i].idade);
    printf("rua: %s\nnmero: %d\n", p[i].rua, p[i].numero);
  }
  return 0;
}

Alocação dinâmica¶

  • A alocao dinmica ocorre quando no sabemos o tamanho de um array em tempo de compilao, sendo necessrio alocar em tempo de execuo.

  • At agora utilizamos alocao esttica para definir o tamanho de um array. Por exemplo, para criar um array de alunos da UFABC, podemos criar um array "muito grande", por exemplo, contendo 100000 alunos.

  • Nesse caso, temos dois problemas, se um dia tivermos mais que 100000 alunos, esse array no vai suportar.

  • O outro problema que em geral estamos desperdiando memria RAM com as posio do array no utilizadas.

  • Para resolver isso, utilizamos alocao dinmica, inserindo ou retirando elementos do array conforme a demanda.

  • Na linguagem C existem os seguintes comandos da biblioteca stdlib.h:

    • malloc
      • sintaxe para array de inteiros: int * p = (int *) malloc( TAMANHO * sizeof(int) )
      • comando utilizado para alocar um array de TAMANHO de inteiros.
      • O ponteiro p aponta para o primeiro elemento do array.
      • possvel definir um array para qualquer tipo de dado, no apenas para int, inclusive para struct.
      • Se p = NULL ento no existe memria suficiente para a alocao.
    • calloc
      • sintaxe para array de inteiros: int * p = (int *) calloc( TAMANHO, sizeof(int) )
      • anlogo ao malloc, porm:
        • o calloc possui dois argumentos;
        • inicializa todos os elementos com zeros;
        • logo, um pouco mais lento.
    • realloc
      • sintaxe para array de inteiros: p = (int *) realloc( p, TAMANHO * sizeof(int) )
      • realoca um array
    • free
      • sempre que alocamos memria dinmicamente devemos liberar a memria no final, com o camando `free(p)`.
      • se isso no for feito, a memria ser liberada somente quando o computador for desligado ou a aplicao encerrada (se o sistema operacional no tiver algum recurso de "coleta de lixo").

Alocação dinâmica para array multidimensional¶

  • possvel usar um array unidimensional para tratar matriz multidimensional. Por exemplo, para uma matriz de L linha e C colunas de inteiros, temos:

    int * m = (int *) malloc( L*C*sizeof(int) );
for (int i=0; i < L; i++) 
  for (int j=0; j < C; j++) 
    scanf("%d", m + i*C +j); // m+i*C+j = &m[i*C+j]

// ou simplesmente
for (int i=0; i < L*C; i++) 
    scanf("%d", m + i); // linha=i/C e coluna=i%C
  • Para utilizarmos matrizes com alocao dinmica interessante trabalharmos com ponteiro para ponteiro:
    • primeiro alocamos memria para as linhas (L) da matriz:
      • int **m = (int**) malloc( L * sizeof(int *) );
    • em seguida alocamos memria para cada linha da matriz (colunas C): 
        for (int i=0; i < L; i++) 
    m[i] = (int*) malloc( C * sizeof(int *) ); // m[i] = *(m+i)
    • para popular a matriz - opo 1: 
        for (int i=0; i < L; i++) 
    for (int j=0; j < C; j++) 
      scanf("%d", &m[i][j]); // &m[i][j] = m + i*C + j
    • para popular a matriz - opo 2: 
        for (int i=0; i < L*C; i++) 
      scanf("%d", m+i); // m+i = &m[i/C][i%C]
    • no esquecer de LIBERAR A MEMRIA:
      • primeiro o contedo de cada linha: 
        for (int i=0; i < L; i++) 
  free(m[i]); // m[i] = *(m+i)
      • depois toda a matriz: 
        free(m);

%%writefile cap8ex02.c
#include <stdio.h>
#include<malloc.h>  

int ** leiaMatriz(int L, int C) {
  int **m = (int **)malloc(L*sizeof(int*)); 
  for (int i = 0; i < L; i++) {
    m[i] = (int *)malloc(C * sizeof(int)); // for each row allocate C ints
    for (int j = 0; j < C; j++) 
      scanf("%d", &m[i][j]); 
  }
  return m;
}
void free_matrix(int **m, int L) {
    for (int i = 0; i < L; i++) 
         free(m[i]);
    free(m);
 }
void escrevaMatriz(int **m, int L, int C) {
  for (int i = 0; i < L; i++) {
   for (int j = 0; j < C; j++) 
      printf("%d\t", m[i][j]); 
   printf("\n");
  }
}
int main(void) {
  // ENTRADA DE DADOS
  int L, C, **m;   // variaveis de referncia m
  printf("Digite o nmero de alunos: ");
  scanf("%d", &L);

  printf("Digite o nmero de avaliaes: ");
  scanf("%d", &C);

  C = C + 1; // a primeira coluna  o RA do aluno

  printf("Digite os elementos da matriz");
  m = leiaMatriz(L,C);

  // PROCESSAMENTO ?

  // SADA DE DADOS
  printf("\nLISTA DE ALUNOS vs Avaliaes:\n");
  printf("RA ");
  for (int i = 0; i < C-1; i++) 
    printf("\t%d",(i+1)); 
  
  printf("\n");
  escrevaMatriz(m,L,C);

  return 0;
}

Acessando elemento com *¶

Outras forma de acessar elementos em matriz.

  for (int i = 0; i < L; i++) 
   for (int j = 0; j < C; j++) {
      printf("%d\t", m[i][j]); // ou
      printf("%d\t", *(m + i*C + j));
   }

Ou utilizando apenas um lao para varrar uma matriz:

  for (int i = 0; i < L*C; i++) {
      printf("%d\t", m[i/C][i%C]); // ou
      printf("%d\t", *(m + i));
   }
Matriz Multimensional¶
for (int k = 0; k < D; k++) // profundidade
  for (int i = 0; i < L; i++) // linha
   for (int j = 0; j < C; j++) { // coluna
      printf("%d\t", m[k][i][j]); // ou
      printf("%d\t", *(m + k*L*C + i*C + j));
   }

Ou utilizando apenas um lao para varrar uma matriz:

  for (int i = 0; i < D*L*C; i++) {
      d = i/(L*C);
      printf("%d\t", m[d][(d-i)/C][(d-i)%C]); // ou
      printf("%d\t", *(m + i));
   }
